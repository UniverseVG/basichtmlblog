Moving on in this article with Python Database Connection let us see the parameters required to connect to the database:

  1. Username- It is simply the username you give to work MySQL server with, the Default username is root.
  2. Password- Password is given by the user when you have installed the MySQL database. I am giving password here as ‘password123
  3. Host Name- This basically is the server name or IP address on which your MySQL is running, If it is a ‘localhost’, then your IP address is 127.0.0.0

I will show you from a coding perspective to connect python with MySQL database.

import mysql.connector
mydb=mysql.connector.connect(host="localhost",user="root",passwd="password123") // I have used 'host','username','password'
print(mydb)
